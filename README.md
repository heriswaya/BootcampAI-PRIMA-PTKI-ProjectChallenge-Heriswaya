# BootcampAI-PRIMA-PTKI-ProjectChallenge-Heriswaya
Repository ini berisi proyek tugas akhir sesi 8 Bootcamp AI PRIMA PTKI. Fokus utama proyek adalah membangun sistem Analisis Sentimen dan Topic Modeling terhadap ulasan e-commerce menggunakan model SentenceTransformer + Logistic Regression untuk klasifikasi sentimen, serta BERTopic untuk pemodelan topik.
